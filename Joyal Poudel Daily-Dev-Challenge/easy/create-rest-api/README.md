# REST API Demo

## Overview

A web application that demonstrates REST API consumption using the JSONPlaceholder API. Users can fetch user information and their posts.

## Features

- ✅ Fetch user details by ID
- ✅ Fetch all posts
- ✅ Fetch posts by user
- ✅ Display raw JSON responses
- ✅ Error handling
- ✅ Loading states
- ✅ Responsive design

## API Endpoints Used

- `GET /users/{id}` - Get user details
- `GET /users/{id}/posts` - Get user's posts
- `GET /posts` - Get all posts

## How to Use

1. Open `index.html` in a web browser
2. Enter a user ID (1-10) and click "Get User" to fetch user details
3. Click "Get User Posts" to fetch posts by the selected user
4. Click "Get All Posts" to fetch all posts
5. View the raw JSON response below

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Fetch API
- JSONPlaceholder (Free REST API)
