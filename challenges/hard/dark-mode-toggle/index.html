Dark Mode Toggle

Toggle between light and dark themes with a smooth, responsive interface.

Why Dark Mode?

Dark mode reduces eye strain in low-light conditions, saves energy on OLED screens, and gives your UI a modern look.

How It Works

This demo uses CSS custom properties for color themes and JavaScript to toggle the theme state and persist the preference.

Try It Out

Click the button above to switch themes. The selected theme is saved so it stays active on page reload.
