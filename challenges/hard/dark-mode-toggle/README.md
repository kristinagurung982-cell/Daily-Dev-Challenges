# Day 1 - Hard Task 5: Dark Mode Toggle

A responsive page with a dark mode toggle that switches between light and dark themes using CSS variables and JavaScript.

## Features

- Light / Dark mode toggle button
- Smooth transition between themes
- Accessible toggle with `aria-pressed`
- Saves theme preference in `localStorage`
- Responsive layout for desktop and mobile

## Files

- `index.html` — page markup and toggle button
- `styles.css` — theme-aware styling and responsive layout
- `script.js` — toggle logic and localStorage persistence
