# Responsive Navigation Bar - Day 1 Challenge

A simple and responsive navigation bar with a logo and three navigation links (Home, About, Contact).

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Logo**: Branded navigation logo
- **Navigation Links**: Home, About, and Contact links
- **Hamburger Menu**: Mobile-friendly toggle menu for smaller screens
- **Smooth Animations**: Transitions and hover effects for better UX
- **Sticky Navigation**: Navigation bar stays at the top while scrolling
- **Smooth Scrolling**: Anchor links navigate to the corresponding sections

## File Structure

- `index.html` - HTML structure with navigation and sample sections
- `styles.css` - Responsive CSS styling with media queries
- `script.js` - JavaScript for hamburger menu functionality

## How to Use

1. Open `index.html` in a web browser
2. Test the navigation by clicking on the links
3. On mobile devices (or when resizing the browser to a smaller width), the hamburger menu will appear
4. Click the hamburger icon to toggle the navigation menu

## Responsive Breakpoints

- **Desktop**: 769px and above - Full navigation menu visible
- **Tablet**: 481px to 768px - Hamburger menu visible
- **Mobile**: 480px and below - Optimized for small screens

## Browser Compatibility

Works on all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge
