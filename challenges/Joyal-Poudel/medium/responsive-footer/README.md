# Responsive Footer

## Overview

A modern, fully responsive footer design with multiple columns, social links, contact information, and smooth animations. Works perfectly on all screen sizes from mobile to desktop.

## Features

- ✅ 4-column footer layout (responsive to 1 column on mobile)
- ✅ Social media links with icons
- ✅ Quick navigation links
- ✅ Services section
- ✅ Contact information with icons
- ✅ Footer bottom with copyright and policy links
- ✅ Smooth animations and transitions
- ✅ Mobile-first responsive design
- ✅ Font Awesome icons
- ✅ Print-friendly styles

## Sections

### About Us

Company information with social media links.

### Quick Links

Navigation to main pages.

### Services

List of services offered.

### Contact Info

Address, phone, and email with icons.

### Footer Bottom

Copyright and policy links.

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- Font Awesome Icons
- Responsive Design

## Breakpoints

- Desktop: Full 4-column layout
- Tablet (768px): Responsive grid
- Mobile (480px): Single column layout
