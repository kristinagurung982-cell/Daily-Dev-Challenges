# Premium Accordion UI

An elegant and interactive accordion system built for modern web applications. This component focuses on smooth height transitions, flexible configuration, and a sophisticated dark aesthetic.

## ✨ Features

- **Liquid Height Animation:** Smooth expansion and collapse effects.
- **Selective Mode:** Toggle between "Single Open" (standard FAQ style) and "Multiple Open" modes.
- **Visual Feedback:** Active items feature unique glow effects and icon rotations.
- **Glassmorphism:** Styled with frosted glass effects and glowing borders.
- **Responsive:** Optimized for both desktop and mobile reading experiences.
- **Zero Dependencies:** Pure Vanilla HTML, CSS, and JS.

## 🛠️ Implementation Details

- **CSS Variables:** Easy branding adaptation.
- **JavaScript State:** Handles complex interaction logic and multi-open toggling.
- **Semantic HTML:** Correct button and panel relationships for better structure.

## 🚀 How to Explore

1. Open `index.html`.
2. Click on the headers to expand the panels.
3. Try toggling the "Allow multiple items open" checkbox at the bottom to change the behavior.
