# Modal Popup

## Overview

A comprehensive modal popup demonstration with three different modal types: basic, confirmation, and form modals. Includes smooth animations and keyboard support.

## Features

- ✅ Basic modal with close button
- ✅ Confirmation modal for user actions
- ✅ Form modal for subscriptions
- ✅ Click outside to close
- ✅ Escape key to close
- ✅ Smooth animations
- ✅ Success/error messages
- ✅ Form validation
- ✅ Responsive design

## Modal Types

### Basic Modal

A simple informational modal with close button and message.

### Confirmation Modal

A modal for confirming important actions with Confirm/Cancel buttons.

### Form Modal

A subscription form modal with name and email fields.

## How to Use

1. Open `index.html` in a web browser
2. Click any of the three buttons to open different modals
3. Close modals by:
   - Clicking the X button
   - Clicking outside the modal
   - Pressing the Escape key
   - Clicking the Close/Cancel button

## Technologies Used

- HTML5
- CSS3 (animations)
- Vanilla JavaScript
- Responsive Design
