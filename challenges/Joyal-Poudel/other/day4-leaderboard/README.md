# Day 4 Leaderboard

A premium, high-fidelity leaderboard component designed to visualize engagement and performance rankings for the Daily Dev Challenges.

## ✨ High-End Features

- **Live Filtering:** Sort by challenge difficulty (`Easy`, `Medium`, `Hard`) in real-time.
- **Search Capability:** Instant developer lookup with live filtering.
- **Visual Hierarchy:** Distinct styling for top 3 positions (Gold, Silver, Bronze levels).
- **Glassmorphism:** Deep frosted glass effect with high-contrast text.
- **Adaptive Avatars:** Automatically generated initial-based avatars with gradients.
- **Responsive Design:** Seamless transition from desktop data grids to mobile-friendly tables.
- **Dynamic Data:** Powered by a clean JavaScript data structure for easy integration.

## 🛠️ Implementation Specs

- **Vanilla JS:** Efficient DOM manipulation for search and filtering.
- **CSS3:** Custom properties for consistent theme management across all Day 4 components.
- **Typography:** Using "Outfit" from Google Fonts for a modern, sleek look.

## 🚀 How to View

1. Open `index.html`.
2. Use the category filters to switch between difficulty levels.
3. Type in the search box to find specific developers.
4. Observe the smooth hover effects and clean table layout.
