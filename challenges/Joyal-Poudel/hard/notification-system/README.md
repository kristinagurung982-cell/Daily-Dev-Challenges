# Global Notification System

A professional, high-performance toast notification engine designed for complex web applications. This system handles multiple notification types, stack management, and automatic dismissal with visual progress tracking.

## ✨ Premium Features

- **Dynamic Stacking:** Notifications stack vertically based on selected placement.
- **Multiple Types:** Dedicated themes for `Success`, `Error`, `Warning`, and `Info`.
- **Progress Tracking:** Integrated progress bars show remaining time before auto-dismissal.
- **Configurable Placement:** Support for all four corners: `Top-Right`, `Top-Left`, `Bottom-Right`, and `Bottom-Left`.
- **Glassmorphism:** Elegant dark theme with frosted glass cards and glowing status borders.
- **Smooth Animations:** Slide and fade transitions powered by CSS Keyframes.
- **Responsive:** Fluid widths that adapt to mobile screens.

## 🛠️ Technical Implementation

- **Class-Based Architecture:** Encapsulated logic for creating and removing notifications.
- **DOM Management:** Dynamic injection and clean removal of elements.
- **State Selection:** Real-time placement switching without page reload.
- **Zero Dependencies:** Pure Vanilla JavaScript implementation.

## 🚀 How to Use

1. Open `index.html`.
2. Use the "Trigger Notifications" buttons to fire off different toast types.
3. Change the "Placement" in the configuration card to see the stacking logic change in real-time.
4. Interact with the close buttons or wait for the auto-dismiss timer.
