# Image Gallery

## Overview

A feature-rich image gallery application with search functionality, multiple view modes, lightbox modal, and keyboard navigation. Responsive and mobile-friendly design.

## Features

- ✅ Grid and Masonry view modes
- ✅ Search/filter images by title or tags
- ✅ Lightbox modal for full-size image viewing
- ✅ Previous/Next navigation buttons
- ✅ Keyboard shortcuts (Arrow keys, Escape)
- ✅ Loading spinner
- ✅ Lazy loading for images
- ✅ Smooth animations and transitions
- ✅ Fully responsive design
- ✅ Touch-friendly on mobile devices
- ✅ Image info display (current/total)

## View Modes

### Grid View

Traditional grid layout with evenly sized thumbnails.

### Masonry View

Pinterest-style masonry layout.

## Controls

### Search

Filter images by entering keywords in the search box.

### View Switcher

Toggle between Grid and Masonry views.

### Lightbox Navigation

- Click on image to open fullscreen view
- Use arrow buttons or arrow keys to navigate
- Press Escape or click outside to close
- Click on heart icon to like images

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- Font Awesome Icons
- Responsive Design
- Keyboard Event Handling
- Local Image Data

## How to Use

1. Open `index.html` in a web browser
2. Browse the image gallery
3. Use the search box to filter images
4. Switch between Grid and Masonry views
5. Click on an image to open the lightbox
6. Use arrow keys or buttons to navigate through images
7. Press Escape to close the lightbox

## Keyboard Shortcuts

- **Arrow Left**: Previous image
- **Arrow Right**: Next image
- **Escape**: Close lightbox

## Notes

- Uses mock image data from Unsplash for demonstration
- Can be integrated with real API endpoints
- Images are optimized for web with lazy loading
