# Premium Loading Spinners

A beautiful collection of modern CSS-based loading animations. This gallery showcases different animation techniques focusing on performance, aesthetics, and smooth motion.

## ✨ Highlights

- **Multiple Variants:** 6 distinct spinner styles (Classic, Orbit, Morphing, etc.).
- **Pure CSS:** No images or heavy libraries; built with keyframe animations.
- **Glassmorphism:** Elegant dark theme with frosted glass cards.
- **Full-Screen Overlay:** Demonstrates how to use a spinner in a real-world scenario (e.g., API loading).
- **Responsive:** Fluid grid layout that works on all devices.

## 🎨 spinner Types

1. **Classic Pulse:** Standard rotating border.
2. **Bouncing Dots:** Playful dots with staggered animation.
3. **Cosmic Orbit:** Planet-like dot orbiting a path.
4. **Modern Bars:** Equalizer-style bars with height modulation.
5. **Glow Circle:** Conic gradient mask for a modern feel.
6. **Morphing Square:** Shape-shifting animation between square and circle.

## 🚀 How to Use

1. Open `index.html` to view the gallery.
2. Click **"Trigger Fullscreen Overlay"** to see a spinner in action as a page-level loader.
