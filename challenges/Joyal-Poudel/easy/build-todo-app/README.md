# To-Do App

## Overview

A simple and efficient to-do application where users can add, complete, and delete tasks. Features local storage persistence to save tasks between sessions.

## Features

- ✅ Add new tasks
- ✅ Mark tasks as complete/incomplete
- ✅ Delete tasks
- ✅ Local storage persistence
- ✅ Clean and responsive UI
- ✅ XSS protection with HTML escaping

## How to Use

1. Open `index.html` in a web browser
2. Type a task in the input field
3. Click "Add Task" or press Enter
4. Click "Done" to mark a task complete
5. Click "Delete" to remove a task

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
