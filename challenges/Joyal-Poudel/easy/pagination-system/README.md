# Premium Pagination System

A smooth, interactive pagination component built for modern web applications. This challenge focuses on creating a responsive and visually appealing way to navigate through large datasets.

## ✨ Features

- **Glassmorphism UI:** Modern dark theme with frosted glass effects.
- **Dynamic Logic:** Automatically handles page numbering, ellipsis (dots), and navigation.
- **Simulated Loading:** Visual feedback during page transitions.
- **Smooth Animations:** Integrated CSS animations for item entry.
- **Responsive Design:** Optimized for mobile and desktop screens.

## 🛠️ Built With

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties, Flexbox/Grid, and Keyframe animations.
- **Vanilla JavaScript:** ES6+ logic without external libraries.

## 🚀 Getting Started

1. Open `index.html` in your browser.
2. Interact with the pagination buttons at the bottom.
3. Observe the smooth transitions and dynamic logic.
