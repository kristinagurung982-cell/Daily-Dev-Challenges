# Ronit-Challenges

Complete solutions for Day 2 Frontend Development Challenges.

## Tasks Overview

1. **Easy Task 1** - Build To-Do App
   - Add, delete, and mark tasks as complete
   - Local storage persistence
   - Clean and intuitive UI

2. **Easy Task 2** - Create REST API
   - Mock API with JSON data
   - GET, POST, PUT, DELETE operations
   - CORS enabled

3. **Medium Task 3** - Build Responsive Footer
   - Mobile-first responsive design
   - Multi-column layout
   - Social media links integration

4. **Medium Task 4** - Create Modal Popup
   - Smooth animations
   - Click outside to close
   - Form validation inside modal

5. **Hard Task 5** - Build Image Gallery
   - Grid layout with responsive design
   - Lightbox functionality
   - Smooth transitions and animations

6. **Leaderboard** - Leaderboard Interface
   - Ranking display
   - User statistics
   - Search and filter functionality

## How to Run

1. Open `index.html` in your browser
2. Navigate to any challenge
3. All challenges use vanilla HTML, CSS, and JavaScript
