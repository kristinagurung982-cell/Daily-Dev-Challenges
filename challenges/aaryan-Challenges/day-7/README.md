# Day 7 Challenges

Welcome to Day 7 of the Daily Dev Challenges! 🚀

This day features five challenges across different difficulty levels and a leaderboard.

## Challenges

### Easy Task 1: Build a URL Shortener Logic
- **Description**: Create a URL shortener that converts long URLs into shortened versions
- **Key Features**: 
  - URL validation
  - Unique short code generation
  - URL-to-short-code mapping
- **Skills**: JavaScript fundamentals, string manipulation, data structures

### Easy Task 2: Implement a Debounce Function
- **Description**: Create a debounce utility function to optimize event handling
- **Key Features**:
  - Function throttling
  - Event optimization
  - Real-time demo with search input
- **Skills**: Higher-order functions, closures, event handling

### Medium Task 3: Create a 3D Card Hover Effect
- **Description**: Build cards with 3D perspective effects on hover
- **Key Features**:
  - CSS 3D transforms
  - Smooth animations
  - Responsive design
- **Skills**: CSS transforms, animations, perspective effects

### Medium Task 4: Build a Markdown to HTML Previewer
- **Description**: Convert markdown text into HTML in real-time
- **Key Features**:
  - Markdown parsing
  - Live preview
  - Support for common markdown syntax
- **Skills**: Text parsing, HTML generation, DOM manipulation

### Hard Task 5: Design a Responsive Dashboard Sidebar
- **Description**: Create a responsive sidebar navigation for a dashboard
- **Key Features**:
  - Collapsible navigation
  - Mobile responsive
  - Multiple menu items and sections
- **Skills**: Responsive design, DOM manipulation, CSS layouts

- Display rankings and scores for Day 7 participants
