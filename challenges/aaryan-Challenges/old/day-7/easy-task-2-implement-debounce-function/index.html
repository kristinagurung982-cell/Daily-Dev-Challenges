What is Debouncing?

Debouncing is a technique to limit the number of times a function is called. It delays the execution until after a specified wait time has passed since the last invocation. Perfect for search inputs, resize handlers, and scroll events.

📝 How Debounce Works

function debounce(func, delay) {
  let timeoutId;
  return function(...args) {
    clearTimeout(timeoutId);
    timeoutId = setTimeout(() => {
      func(...args);
    }, delay);
  };
}

Use Cases:

  • Search input validation and API calls
  • Window resize event handlers
  • Scroll event listeners
  • Auto-save functionality
  • Form field validation
