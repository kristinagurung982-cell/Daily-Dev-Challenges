# Day 6 Challenge Solutions

This folder contains complete solutions for Day 6 issues 1 through 6.

## Tasks

1. **Easy Task 1**: Build To-Do App - A fully functional to-do application with add, delete, and complete functionality with localStorage support.
2. **Easy Task 2**: Create REST API - A user management system with create, read, delete operations using localStorage as a backend.
3. **Medium Task 3**: Build Responsive Footer - A modern, responsive footer with multiple sections, social links, and mobile optimization.
4. **Medium Task 4**: Create Modal Popup - Multiple modal windows with open/close functionality, keyboard support, and form integration.
5. **Hard Task 5**: Build Image Gallery - Advanced image gallery with search, filtering, lightbox view, and keyboard navigation.
6. **Day 6 Leaderboard**: Leaderboard interface with rankings, sorting capabilities, and statistics.

## How to Run

1. Open `index.html` in your browser to see the overview page
2. Click on any task card to view the solution
3. Each solution is fully functional and interactive
4. Check the leaderboard to see rankings

- ✅ Fully responsive design for all devices
