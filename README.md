# 🤝 Contributing Guide

Thanks for contributing! 🚀

## Steps:

1. Fork this repo
2. Clone your fork
3. Pick an issue
4. Solve the task
5. Push your code
6. Create a Pull Request

---

## Rules:

- One issue per day
- Follow folder structure
- Write clean code
- Add comments if needed

---

Happy Coding 💻🔥
